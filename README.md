# Plataforma Pynpon

Pynpon es una aplicación web conformada por 3 sistemas que se relacionan entre sí.

1. Centro de Compras 
2. Centro de Ventas (Central de Ventas)
3. Sistema Administrativo (Administración y Gestión de la plataforma)
